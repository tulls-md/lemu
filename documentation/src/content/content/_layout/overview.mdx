# Шаблон

Шаблон (`Layout`) задаёт каркас страницы и отвечает за компоновку её основных
зон. Он определяет, как располагаются навигация, содержимое и вспомогательные
элементы, обеспечивая единообразную структуру интерфейса между экранами.

Шаблон состоит из набора компонентов, каждый из которых отвечает за свою часть
макета и может использоваться как вместе, так и по отдельности.

## Root

Корневой контейнер макета. Объединяет основные зоны страницы — шапку, контент и
подвал — и задаёт общую структуру и компоновку интерфейса.

## Header

Шапка страницы — верхний навигационный блок макета. Разделена на три зоны
(слева, по центру, справа) и обычно содержит заголовок, навигацию и действия.

`Header` размещается внутри `Root` как верхняя зона макета.
