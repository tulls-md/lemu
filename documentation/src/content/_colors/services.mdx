# Сервисные цвета

Семантические палитры для передачи статуса интерфейса: подтверждение
успешного действия, предупреждение и ошибка. Они построены по той же
12-шаговой шкале, что и [основные палитры](./colors), и используют те же
вспомогательные токены (`*-a*`, `*-contrast`, `*-surface`, `*-indicator`,
`*-track`).

Применяйте сервисные цвета осознанно — только для передачи статуса, а не в
качестве декоративных акцентов.
