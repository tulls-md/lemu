# Цвета

Палитра CSS-токенов цветов, используемых в дизайн-системе Lemu. Палитры
сгенерированы на основе [Radix Colors](https://www.radix-ui.com/colors) и
подключаются как CSS-переменные на уровне `:root`.

Каждая палитра автоматически адаптируется под широкий цветовой охват: на
дисплеях с поддержкой P3 (`@media (color-gamut: p3)`) значения переопределяются
в пространстве `oklch`, обеспечивая более насыщенное и точное отображение.

## Анатомия палитры

Каждая цветовая палитра построена по единой 12-шаговой шкале. Шаги имеют
устойчивое назначение, поэтому один и тот же шаг можно переиспользовать для
одинаковых задач в разных цветах.

| Шаги | Назначение |
| --- | --- |
| 1–2 | Фон страницы и приложения |
| 3–5 | Фон компонентов (обычный, наведение, активный) |
| 6–8 | Границы и разделители |
| 9–10 | Заливка акцентных элементов (обычная и при наведении) |
| 11–12 | Текст (вторичный и основной) |

Помимо основной шкалы каждая палитра содержит вспомогательные токены:

- **`*-a1` … `*-a12`** — полупрозрачные (alpha) версии шагов для наложения на
  произвольный фон.
- **`*-contrast`** — цвет текста, читаемый поверх заливки (шаг 9).
- **`*-surface`** — полупрозрачный фон для поверхностей и подложек.
- **`*-indicator`**, **`*-track`** — цвета для индикаторов и треков (слайдеры,
  прогресс-бары и т. п.).
