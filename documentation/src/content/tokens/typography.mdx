# Типографика

Типографическая шкала CSS-токенов, на которой строится весь текст в дизайн-системе
Lemu. Токены задают единый ритм размеров, насыщенности и межстрочных интервалов и
подключаются как CSS-переменные на уровне `:root`.

Опирайтесь на эти токены вместо произвольных значений `font-size`, `font-weight` и
`line-height`. Общая шкала обеспечивает визуальную согласованность интерфейса,
предсказуемый вертикальный ритм и упрощает поддержку: изменение одного токена
синхронно отражается на всех ролях, которые его используют.

## Анатомия

Токены разделены на два уровня:

- **Примитивы** — атомарные значения: семейство (`--family-*`), размеры
  (`--text-size-*`), насыщенность (`--weight-*`) и межстрочные интервалы
  (`--line-height-*`). Имя токена прямо называет свойство, за которое он отвечает.
- **Роли** — готовые `font`-шорткаты (`--font-*`), собранные из примитивов под
  конкретную задачу: заголовок, основной текст, подпись, текст кнопки. Роль
  применяется одним свойством `font`, поэтому в разметке достаточно одной строки.

В повседневной работе используйте роли. К примитивам обращайтесь, когда нужно
донастроить отдельное свойство, которое роль не покрывает.

Основное семейство — [Golos Text](https://fontsource.org/fonts/golos-text),

Размеры заданы в `rem` и масштабируются вместе с корневым размером шрифта.

## Роли

Роли — это собранные `font`-шорткаты для типовых элементов интерфейса. Применяйте
их одним свойством:

```css
.title {
  font: var(--font-h2);
}
```
