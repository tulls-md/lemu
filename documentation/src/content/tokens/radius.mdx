# Скругления

Шкала CSS-токенов скругления углов, используемых в дизайн-системе Lemu. Токены
задают единый радиус скругления для компонентов и подключаются как
CSS-переменные на уровне `:root`.

Используйте эти токены для свойства `border-radius` всех элементов: кнопок,
полей ввода, карточек, модальных окон и других компонентов. Опора на общую
шкалу вместо произвольных значений обеспечивает визуальную согласованность
интерфейса и упрощает поддержку.

## Анатомия шкалы

Шкала построена на базовом шаге **2px** и растёт линейно. Имя токена содержит
множитель базового шага: `--radius-{n}x` равен `n × 2px`. Например,
`--radius-4x` — это `4 × 2px = 8px`.

Линейный шаг даёт предсказуемую и плотную шкалу: соседние значения отличаются
ровно на базовый шаг, поэтому подобрать соразмерный радиус можно без расчётов.

Помимо числовой шкалы доступны два специальных токена:

- **`--radius-full`** — практически бесконечный радиус (`9999px`) для
  «таблеток» (pill) и полностью скруглённых концов прямоугольных элементов.
- **`--radius-circle`** — `50%` для превращения квадратного элемента в круг
  (аватары, иконочные кнопки, индикаторы).
